Rock Paper Scissors Game

An interactive Rock-Paper-Scissors web application built using HTML, CSS, and JavaScript, featuring image-based gameplay, real-time results, and dynamic score tracking.

Features :
1. Image-based user interaction (Rock, Paper, Scissors)
2. Randomized computer moves
3. Instant result display (Win / Lose / Draw)
4. Real-time score tracking
5. Responsive and user-friendly UI
6. Hover effects and smooth transitions

Tech Stack:
HTML5 – Structure
CSS3 – Styling & responsiveness
JavaScript (ES6) – Game logic & DOM manipulation

⚙️ How It Works
User selects Rock, Paper, or Scissors by clicking an image
Computer randomly selects a move
JavaScript compares both choices
Result is displayed instantly
Score updates dynamically
